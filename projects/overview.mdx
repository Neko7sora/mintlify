## 使い分け

| リポジトリ | 主な用途 | 技術スタック |
| --- | --- | --- |
| `discord-tauri` | Discord をデスクトップアプリとして扱う | Tauri、Vite、TypeScript |
| `misskey-tauri` | Misskey をデスクトップアプリとして扱う | Tauri、Vite、TypeScript |
| `Discord-Oauth2` | Discord OAuth2 の認証コード交換を試す | Node.js、Express、dotenv |
| `patches` | npm 依存関係へローカルパッチを適用する | patch-package |

## 読み始める順番

1. Tauri アプリの構成を知りたい場合は、[`discord-tauri`](/projects/discord-tauri) または [`misskey-tauri`](/projects/misskey-tauri) から読みます。
2. Discord OAuth2 の流れを確認したい場合は、[`Discord-Oauth2`](/projects/discord-oauth2) を読みます。
3. 依存パッケージへの修正を共有したい場合は、[`patches`](/projects/patches) を読みます。
