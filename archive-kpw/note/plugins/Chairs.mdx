## Permissions

- chairs.reload
  <br />
  `/chairs reload `- Reload the Chairs configuration file.

- chairs.sit
  <br />
  Sit down on chairs.

- chairs.sit.health
  <br />
  If the health sitting effects are enabled then players with this permission node are healed while siting.
